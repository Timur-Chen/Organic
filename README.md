# 🌿 Organick - Organic Food Website

A modern and responsive **organic food landing page** built using **HTML, SCSS, and JavaScript**.
This project represents a clean and structured frontend implementation of an organic products website.

---

## Features

* Responsive modern UI
* Organic products section
* Testimonials slider
* Product cards
* Navigation menu (desktop + mobile)
* Newsletter subscription section
* Clean SCSS architecture
* Reusable UI components

---

## Technologies Used

* HTML5
* CSS3
* SCSS (Sass)
* JavaScript
* Swiper.js
